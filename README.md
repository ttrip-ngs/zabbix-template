# Zabbix Templates

このリポジトリはZabbix用のテンプレート等を管理するリポジトリです。

## 概要

Zabbix 7.0 LTS向けのカスタムテンプレートを管理しています。

## ディレクトリ構成

参考リポジトリ: [zabbix/community-templates](https://github.com/zabbix/community-templates)

## テンプレート形式

- 対応Zabbixバージョン: 7.0 LTS
- ファイル形式: XML

## 開発フロー

各テンプレート作成時は以下の手順で進めます:

1. 新規featureブランチを作成
2. テンプレートを開発
3. Zabbix上で動作確認を実施
4. 確認完了後、mainブランチへマージ
